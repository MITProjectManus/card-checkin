# Makerspaces Card Tap Checkin
The next version of a card tap checkin program in Python able to run on Raspberry Pis (and other hardware) and process MIT ID card taps (and digital ID taps) to check users into and out of makerspaces. This replaces [Airtable-Card-Checkin](https://github.com/MITProjectManus/Airtable-Card-Checkin), the original prototype version of card tap checkin for MIT makerspaces.

## env.py

The `env.py` file contains API keys and location-specific configuration items that should **NOT** be tracked in Git. This file is excluded via `.gitignore`.
